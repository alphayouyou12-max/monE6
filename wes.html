Contexte professionnel :
Dans le cadre de mon stage chez SFR (Agence Régionale), le service des Ressources Humaines rencontrait des problèmes de sécurité et de gestion de parc. Les postes étaient en groupe de travail (Workgroup), l'adressage IP était manuel (source de conflits), et aucune stratégie de sécurité centralisée n'était appliquée. Il était nécessaire de migrer vers une infrastructure centralisée pour sécuriser les données sensibles (contrats, paies).
Intitulé de la réalisation professionnelle :
Déploiement d'un Contrôleur de Domaine Windows Server 2022 avec services d'infrastructure (DNS/DHCP) et sécurisation par Pare-feu.
Période de réalisation : Du 17/12/2024 au 13/01/2025 Lieu : CFA INSTA / SFR
Modalité : Seul(e) x En équipe
Compétences travaillées :
  • x Administrer des systèmes informatiques centralisés (Installation AD DS, promotion Windows Server).
  • x Gérer des identités et des accès (Création comptes utilisateurs RH, Groupes, UO).
  • x Configurer des services réseau (DHCP pour l'adressage dynamique, DNS pour la résolution).
  • x Protéger les données et l'identité (Configuration du Pare-feu Windows Defender, GPO de base).
Description de la réalisation (Démarche, acteurs, moyens) :
L'objectif était de créer un domaine sfr.local pour centraliser la gestion.
1. Installation Serveur : Installation de Windows Server 2022 sur machine virtuelle. Configuration IP fixe (192.168.10.200).
2. Active Directory : Installation du rôle AD DS, promotion en Contrôleur de Domaine. Création d'une UO "Ressources_Humaines" et des utilisateurs.
3. Services Réseau :
  - DHCP : Création d'une étendue (Scope) 192.168.10.20-50 pour servir les PC du VLAN 10.
  - DNS : Vérification des zones directes et inversées.
4. Sécurité : Configuration du Pare-feu Windows Defender pour filtrer les flux entrants.
5. Intégration : Jonction du poste client (Windows 10) au domaine et test d'ouverture de session.
Description des ressources utilisées :
Matériel/Logiciel : Serveur Windows 2022 (VM), Client Windows 10 (VM), VMware Workstation (LAN Segments), Switch virtuel.
Documentation : TechNet Microsoft (Rôles AD/DHCP), Plan d'adressage IP SFR fournie par le tuteur.
